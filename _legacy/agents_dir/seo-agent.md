## Инструменты

### Ahrefs MCP (подключён)
Используй Ahrefs MCP-сервер для анализа. Доступные возможности:
- **Keyword Explorer** — анализ ключевых слов, объём поиска, сложность
- **Site Explorer** — анализ домена, органический трафик, топ-страницы
- **Backlink Profile** — анализ бэклинков, ссылочная масса, DR/UR
- **Content Explorer** — поиск популярного контента по теме
- **Rank Tracker** — мониторинг позиций по ключевым словам

### Задачи с Ahrefs
1. **Анализ ключевых слов**: проверь объём и сложность ключевых запросов для KK и RU
2. **Анализ конкурентов**: найди конкурентов (shejire.kz, ata-meken.kz и т.п.), изучи их топ-страницы
3. **Бэклинки**: проверь текущий ссылочный профиль skezire.kz, найди возможности для линкбилдинга
4. **Content Gap**: найди запросы, по которым ранжируются конкуренты, но не мы
5. **Аудит сайта**: проверь технические SEO-проблемы (broken links, redirects, скорость)

## Техническое SEO — чеклист

### Метатеги (для каждой страницы)
1. `<title>` — уникальный, 50-60 символов, ключевое слово в начале
2. `<meta description>` — уникальный, 150-160 символов, с призывом к действию
3. `<meta keywords>` — 5-10 релевантных слов
4. Open Graph (og:title, og:description, og:image, og:url, og:type)
5. Twitter Card (twitter:card, twitter:title, twitter:image)
6. Canonical URL
7. Hreflang (kk, ru, x-default)

### Структурированные данные (JSON-LD)
1. **Главная**: WebSite, Organization, FAQPage
2. **Статьи/блог**: Article (headline, datePublished, dateModified, author, image, publisher)
3. **Энциклопедия**: BreadcrumbList, Article, Person, Place
4. **Глоссарий**: DefinedTermSet, DefinedTerm
5. **Листинги**: ItemList, CollectionPage

### Технические требования
1. sitemap.xml — все URL обоих языков, актуальный lastModified
2. robots.txt — Allow: /, Disallow: /_next/, Sitemap ссылка
3. Заголовки H1-H6 — один H1 на страницу, ключевое слово в H1
4. Alt-тексты для изображений
5. Быстрая загрузка (Core Web Vitals)
6. Мобильная адаптация
7. HTTPS

### Верификация
- Google Search Console — верифицирован (meta tag в layout.tsx)
- Ahrefs — подключён через MCP
